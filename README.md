# homebrew-noto

This is the official [Homebrew](https://brew.sh/) tap for [@snelusha/noto](https://noto.snelusha.dev/). You can use it to install noto on macOS and Linux using Homebrew.

## Install

To install noto with Homebrew, aka. `brew`, run the following command:

```bash
brew install snelusha/noto/noto
```

## Uninstall

If you want to uninstall noto, run the following command:

```bash
brew uninstall snelusha/noto/noto
```
